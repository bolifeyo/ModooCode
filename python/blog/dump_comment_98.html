    argc, argv 라는 이름은 미리 정해진 이름인지, 우리 맘대로 다르게 지을 수 있는 변수명인지 궁금합니다.

    2013.12.04 11:13 신고 [ ADDR : EDIT/ DEL : REPLY ]
    • Psi

      1. 프로그램이 종료되면 운영체제가 알아서 반환해줍니다. 하지만 우리가 꼭 free 를 해주라는 이유는, 위 말을 다시 말하면 프로그램이 종료되기 전 까지 반환하지 않은 메모리는 반환되지 않게 되므로, 프로그램 실행시에 메모리 낭비가 됩니다. 흔히, 어떤 프로그램이 이상할 정도로 메모리를 많이 쓰는 경우가 있는데 이를 가리켜 메모리 누수(memory leak) 이라고 하며 위와 같이 메모리 반환을 제때 안해서 메모리가 낭비되는 경우를 말합니다.

      2. 표준은 int argc, char** argv 를 인자로 넣는 것이 원칙이지만 그냥 () 해도 됩니다. 물론 인자 이름은 맘대로 해도 되지요.

      int a, char** b 해도 됩니다.
