    위에 예제중에서요

    이해안되는게있어서 질문드릴게있는대

    가운대부분에 예제에서
    char copy_str(char *dest, char *src)
    {
    while (*src)
    {
    *dest = *src;
    src ++;
    dest ++;
    }

    *dest = '\0';

    return 1;
    }

    이부분이 이해가 안되요
    while문을 왜쓴지와 내용을 대입해서 넣은거 까지는 이해가는대
    char형인대 ++연산자를쓰신거 그리고 리턴을 1로아신게 이해안가요
    혹시 ++연산자부분을 배열 a[1]=b[1] 이런씩으로 한글자식 대입한다고 while문을 쓰시고
    복사하는내용을 다 배열에 집어넣으면 while문이 종료하시게 하는건가요?
    그리고 마지막 '\0'은 문자열 마지막에 넣으신거구요?
    그리고 리턴을 왜 1로 했는지가 제일 궁금해요
