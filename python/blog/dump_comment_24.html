    • 오리뎅이

      배열 이름은 배열의 첫번째 원소를 가르키는 포인터입니다.
      배열의 포인터 변수를 선언하는 방법은 아래와 같습니다.



      int a[3] = { 1, 2, 3];
      int *pA;
      int (*pA1)[3];
      int **pPI;

      pA = a; // a는 백열의 첫번째 원소 &a[0]를 가르키는 포인터, type이 int, 크기 4
      pA1 = &a; // &a는 배열 a[3] 전체를 가르키는 포인터, type이 int [3], 크기 12
      pPI = &a; // 이것은 포인터 type이 int **과 int (*)[3]으로 서로 다르기때문에 안되는 것입니다. int(*)[3]도 이중 포인터입니다.

      pA1 = &a // 여기서 pA1이 가르키는 것은 a인데, a는 int * type입니다. pA1이 배열 전체(사이즈 12bytes)를 가르키기 때문에 pA1++을 하면 주소가 12 증가합니다. 만일 pA1이 int ** type이었다면 pA1++하면 가르키는 곳의 자료형이 * 이기때문에 4byte 증가하게 됩니다. 포인터 type이 이렇게 다를 경우 주소 연산의 결과가 달라지기 때문에 type이 맞지 않는 경우, 대입 연산을 하려고 하면 컴파일 에러가 나는 것입니다. 물론 개발자가 그런 내용을 모두 알고 그렇게 사용하고자 한다면, int ** type으로 캐스팅 해 주면 컴파일이 됩니다.

      좀더 이해를 돕기 위해 2차원 배열도 예를 들어 보겠습니다.

      char b[2][2] = { { 1, 2}, };
      char *pB;
      char (*pB1)[2];
      char (*pB2)[2][2];

      pB = b[0]; // b[0]는 &b[0][0]를 가르키는 포인터, type이 char, 크기 1
      pB1 = b; // b는 b[0]를 가르키는 포인터, type이 char [2], 크기 2
      pB2 = &b; // &b는 배열 전체를 가르키는 포인터, type이 char [2][2], 크기 4

      배열 포인터를 선언하는 방법은 위의 예제와 같이 배열명 자리에 괄호안에 *와 포인터 변수를 넣어 주면 됩니다.

      int (*ptr)[3][3][3];
