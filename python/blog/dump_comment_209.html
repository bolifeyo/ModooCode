    좋은 강의 감사합니다!!

    저같은 경우는 주신 코드대로 짜니까 출력은 되는데 에러가 계속 나서 employee_list랑 manager_list를 Employee와 Manager class에 대한 더블 포인터가 아니라 포인터로 선언을 해주고 그에 맞게 코드를 고쳤는데 잘 돌아가네요~

    좀 궁금했던 부분이 더블포인터는 일종의 2차원배열 같은 건데 employee 리스트나 manager리스트를 궂이 2차원 배열로 선언해줘야 할 이유가 있나요? 아래 코드 첨부합니다.


    class EmployeeList {

    int alloc_employee;
    int current_employee;
    int current_manager;
    Employee * employee_list;
    Manager * manager_list;

    public:

    EmployeeList(int alloc_employee,int alloc_manager) : alloc_employee(alloc_employee) {
    employee_list = new Employee [alloc_employee];
    manager_list = new Manager [alloc_manager];
    current_employee = 0;

    }

    void add_employee(Employee employee) {
    employee_list[current_employee] = employee;
    current_employee++;
    }

    void add_manager(Manager manager) {
    manager_list[current_manager] = manager;
    current_manager++;
    }

    int current_employee_num() {
    return current_employee;
    }
    int current_manager_num() {
    return current_manager;
    }
    void print_employee_info() {
    int total_pay = 0;
    for (int i = 0; i < current_employee; i++) {
    employee_list[i].print_info();
    total_pay += employee_list[i].calculate_pay();
    }
    for (int j = 0; j < current_manager; j++) {
    manager_list[j].print_info();
    total_pay += manager_list[j].calculate_pay();
    }
    cout << "Total amount of pay : " << total_pay<<"만원"<< endl;
    }

    ~EmployeeList() {

    delete[] employee_list;
    delete[] manager_list;

    }

    };

    int main() {
    EmployeeList emp_list(7,7);
    emp_list.add_manager(Manager("Jaehun Kim", 24, "회장", 10,19));
    emp_list.add_manager(Manager("Peter Kim", 23, "회장 위의 권력자", 100,2));
    emp_list.add_manager(Manager("nicholas Kim", 23, "총무", 8,23));
    emp_list.add_manager(Manager("Ryung-teak O", 21, "부회장", 5,16));
    emp_list.add_employee(Employee("Hye-Jin Lee", 21, "평회원", 2));
    emp_list.add_employee(Employee("Jiho Kim", 20, "평회원", 2));
    emp_list.add_employee(Employee("Yeon-seo O", 24, "평회원", 5));
    emp_list.print_employee_info();

    return 0;
    }
