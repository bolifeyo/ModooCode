    int arr[2][]이 안되는 이유로, int arr[2][3] 인지, int arr[2][4]인지 알 수 없기 때문이라고 설명하셨는데요.

    저는 int arr[2][]가 안되는 이유를 혼자서 다음과 같이 생각했었습니다.
    arr[][3]인 경우는 컴퓨터가 arr[2][3]인지, arr[3][3]인지, 즉 한차원만 검토하면 되지만, arr[2][]의 경우에는 컴퓨터가 arr[0]의 배열이 3이고, arr[1]의 배열은 4로 인식할 수 있다고 보면, arr[2][]에서 []부분을 어느 배열을 기준으로 설정해야 하는지 모르기 때문이라고 생각을 잠시 했었는데, 잘 모르겠네요.
