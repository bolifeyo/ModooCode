    • Psi

      뭐 다 가능한 것들입니다.

      예를 들어 위의 경우는 아래 코드를 보면서 생각해보시고요,
      #include <stdio.h>
      struct STRUCT
      {
      int a;
      int* ptr;
      };
      int main()
      {
      struct STRUCT wow;
      int c;

      wow.ptr = &c;

      *wow.ptr = 3;

      printf("%d", c);

      return 0;
      }

      그 다음 것은 아래 코드를 보면서 생각해보세요

      #include <stdio.h>
      struct STRUCT
      {
      int a;
      struct STRUCT* ptr;
      };
      int main()
      {
      struct STRUCT wow, yeah;

      wow.ptr = &yeah;

      /* 아래 문장의 의미는, wow 의 멤버 중 ptr 이라는 멤버가 가리키는
      구조체의 멤버 a 의 값을 3 으로 바꾼다는 것입니다.
      이 때, wow 의 멤버 ptr 은 현재 yeah 라는 구조체를 가리키고 있으므로
      wow.ptr->a = 3 을 하게 되면 yeah 구조체의 멤버 a 의 값을 3 으로 변경
      하게 됩니다. */
      wow.ptr->a = 3;

      printf("%d", yeah.a);


      return 0;
      }

      아마도 코드를 보시면서 생각해 보시면 금세 답을 구할 수 있으실 것입니다.
