      맞아요 4자리씩 끊어서 읽으면!
      11001010(2)를 4자리씩 끊어서 읽으면
      1100 1010 이 됩니다.
      1100을 예로들면

      d c b a
      1 1 0 0

      각각의 숫자 위에 위 텍스트처럼 알파뱃이 있다고 합시다.
      이 알파뱃은 a부터 2의 0승, b는 2의 1승, c는 2의 2승, d는 2의 3승
      이런식입니다. 즉 c와 d가 1이므로 c는 2의2승이므로 4이고, d는 마찬가지로 2의 3승이 됩니다. 두수를 더하면 12가 되죠.
      같은 방법으로 1010은 10이 됩니다.

      앞의 1100은 12이며 16진수로 C입니다.
      뒤의 1010은 10이며 16진수로 A입니다.

      즉.... 16진수로 바꾸면 CA입니다-.-ㅋ
