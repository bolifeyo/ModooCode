  18. 한창엽

    헠... C중에서 EOF 쓰는 것(Ctrl+z)에 대해 질문인데요,

    뭘 하고 있었냐면

    값들을 입력하다가(반복문 while써서)
    Ctrl+z를 누르면 입력을 종료하고 입력된 값들 중에서 최솟값을 출력하는 예제가 있었어요.

    그런데 터미널 단축키 중에 Ctrl+z가 일시정지 단축키로 겹치는 듯한;;


    이럴 땐 어카죠;

    #include <stdio.h>
    #include <limits.h>
    int main(void)
    {
    int number, min_value = INT_MAX;

    printf("정수를 입력하시오\n종료는 Ctrl + z \n");

    /*******************************************************************************
    Find out the minimum number among the user input, before he or she presses 'Ctrl + Z'.
    *******************************************************************************/

    while(scanf("%d", &number) != EOF) //If 'EOF' did not input, repeat the following 'while' sentence.
    {
    if(number < min_value)
    min_value = number; //If the number user put is smaller than the previous value, change 'min_value' to current 'number'.
    }

    printf("최솟값은 %d\n", min_value);

    return 0;
    }



    --------------------------------------------------------------------------------------------------------------------

    chang-yeop@chang-yeop:~$ gcc -o min min.c
    chang-yeop@chang-yeop:~$ ./min
    정수를 입력하시오
    종료는 Ctrl + z
    10
    20
    30
    5
    ^Z
    [1]+ Stopped ./min
    chang-yeop@chang-yeop:~$
