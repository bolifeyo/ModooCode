iosbase::getloc

locale getloc() const;

현재 로케일(locale) 을 알아낸다.스트림 객체의 로케일을 리턴한다.

 인자

없다.

 리턴값

스트림의 로케일 객체를 리턴한다.

 실행 예제

// 로케일 이름을 출력한다.
#include <iostream>
#include <locale>
using namespace std;

int main() {
  locale current = cout.getloc();
  cout << current.name() << endl;  // 현재 로케일 이름을 출력한다.
  return 0;
}
